Welcome to Advolition

Start Game

Advolition is a Roughlike game in which 1-4 players program robots to explore a dungeon and collect items that contribute to their power and wealth.

The robots are programmed in Javascript.

Programming your robot

Each robot is programmed by creating a js file that describes a new JavaScript class. The new class also needs to extend the Program class and contain a run function that accepts three variables: myID, myData, and myAction and returns myAction. Finally, it's necessary to create a new unknamed object from your class at the end of the script.

The following is an example of a minimal robot script.

            class MyProgram extends Program {
                run(myID, myData, myAction) {             
                    return myAction;
                }          
            }            
            new MyProgram();
